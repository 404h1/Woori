## Part 2. 1등 공식 — 다른 대회 1등 패턴 추출

### 공식 1. **"구체적 시나리오 한정"** (KB Future-9th 우승)
- KB 2024 우승작: **"딥페이크 신분증 위조 탐지"** — 막연한 "AI 사기방지"가 아니라 **신분증 위조** 한 시나리오에 집중
- **한마디 적용:** "시니어 가스라이팅 보이스피싱 통화 중 lock" — 한 시나리오에 집중

### 공식 2. **"특정 사용자 세그먼트"** (신한 SOL 챌린지)
- 신한 우승작: **"외국인 노동자 송금 안전"** — 모든 사용자 X, 외국인 노동자 한 세그먼트
- **한마디 적용:** 시니어 + 시각·청각·발달·인지 장애인 — 명확한 취약계층 세그먼트

### 공식 3. **"실제 도입 가능 PoC"** (하나 디지털 해커톤)
- 하나 우승작: **"악성앱 자동 탐지 + 송금 차단"** — 실제로 하나원큐에 반영, **송금 보이스피싱 0건** 달성
- **한마디 적용:** 6개월 M6 시니어플러스 베타 출시. 우리WON뱅킹 인프라 위에서 즉시 PoC 가능

### 공식 4. **"멀티모달"** (과기정통부 AI Grand Challenge)
- ETRI 우승작: **"실시간 통화 STT + LLM 의도분류"** — 음성+텍스트+의도 동시 처리
- 카이스트 2위: **"화자 인식 + 딥보이스 통합"** — 다층 멀티모달
- **한마디 적용:** 음성(Wake-word) + 버튼 + 흔들기 + 그림카드 4중 트리거

### 공식 5. **"가족·신뢰인 연계"** (코리아핀테크위크 류)
- 핀테크 수상 트렌드: **"시니어 위임송금 + 가족 승인"** 류 콘셉트가 반복 수상
- **한마디 적용:** Trusted Contact 디지털화 + SECURE 영상통화 + 안부 한마디 일상 운영

### 공식 6. **"한 화면 데모 임팩트"** (모든 우승작 공통)
- 우승작은 PT 데모가 30초~1분에 핵심 메시지 시각화 가능
- **한마디 적용:** 할머니 "도와줘!" → LOCK + 자녀 영상통화 발신 = 30초 데모

### 공식 7. **"학계 → 산업 2년 갭 노리기"** (07_competitor_ideas.md)
- 2024 학계 (Comprehension Verification, Trusted Contact 디지털화, 멀티모달 IDS) → 2026~27 산업 적정 시점
- **한마디 적용:** FINRA Rule 4512(2018) + Trusted Contact 연구 + 멀티모달 접근성 = 2026 적기

→ **한마디는 7개 공식 모두 충족하는 유일한 아이디어.**

## Part 4. PT 발표에서 평가위원 마음을 잡는 7가지

심사위원은 우리은행 금융소비자보호그룹·금감원 출신·학계 전문가일 가능성 큼. 이들이 듣고 싶어하는 7가지:

1. **"우리 회사 정체성과 닿아 있는가"** → 시니어플러스 + 사전예방적 소비자보호 + 이사회 금소위 ✅
2. **"법적 기반이 단단한가"** → 금소법 + 무과실 배상제 + FINRA Rule 4512 ✅
3. **"숫자로 증명되는가"** → 무과실 2,800억, ELS 1.4조, 시니어 1,700만원 ✅
4. **"학술 근거가 있는가"** → 노벨경제학상 11개 논문 평균 1만+ 인용 ✅
5. **"기존 한계를 극복하는가"** → 사후 FDS·옵트인 가입의 한계 명확히 짚음 ✅
6. **"바로 도입 가능한가"** → 우리은행 자산 활용 + 6개월 M6 베타 ✅
7. **"감동이 있는가"** → 할머니 "도와줘!" 시연 ✅
