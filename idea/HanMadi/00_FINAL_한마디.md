## 2. 해결책 — "한마디"

### 2-1. 핵심 메커니즘 (4중 트리거)

| 트리거 | 대상 | 동작 |
|---|---|---|
| **음성**: "도와줘", "이상해", "사기야" (사용자 사전 등록) | 일반 시니어 | 온디바이스 Wake-word + LLM 시나리오 분류 |
| **버튼**: WON뱅킹 화면 상단 항상 표시되는 큰 빨간 SOS | 시각·청각·인지 다중 | 한 번 탭 |
| **흔들기**: 폰 3번 흔들기 (3초 내) | 시각장애·말 못함 | 가속도계 패턴 인식 |
| **그림카드**: 4장 (도움/사기/혼란/긴급) | 발달장애·인지저하 | 한 장 탭 |

### 2-2. 발동 시 (3초 이내 자동 시퀀스)

```
[T+0초]  사용자가 트리거 발동
[T+1초]  ┌─ 우리WON뱅킹 모든 송금·청약 5분 즉시 LOCK
         ├─ 사전 등록 가족에게 SECURE 영상통화 자동 발신
         ├─ AI 위시퍼가 "어떤 상황이세요?" 음성으로 질문
         └─ 위치 정보 + 최근 거래 + 통화 발신자 자동 패키지화
[T+3초]  AI 시나리오 분류:
         ├─ 보이스피싱 의심 → 통합대응단(112) 자동 신고
         │                  → LOCK 12시간 연장
         │                  → 발신자 번호 ASAP 플랫폼 자동 등록
         ├─ 인지혼란 → 가족 우선 연결 + 영업점 방문 안내
         ├─ 사기 진행 중 → 사기범 번호 자동 추적 + 통화 즉시 종료
         └─ 단순 도움 → AI 챗봇 1차 응대 → 필요 시 사람 상담
```

### 2-3. 평소 운영 (배경 보호)
- **'안부 한마디'** — 자녀가 부모에게 매일 정해진 시간 짧은 안부. 부모 응답 없으면 자녀에게 알림.
- **'잠금 시간대'** — 시니어 본인이 "밤 10시~아침 6시 송금 자동 차단" 설정.
- **'가족 그룹'** — 우리WON뱅킹 안에 가족 SECURE 채널 (E2E 암호화). 외부 통신망 음성 사칭 우회.

- 자녀(연구원 1명) 화면 등장: "엄마, 무슨 일이야?"
- AI 위시퍼: "방금 검찰 사칭 통화가 감지되었어요. 안전을 위해 5분간 송금이 차단됩니다."
- 동시에 화면 하단: 통합대응단 자동 신고 완료 표시

**② 시각장애 시나리오 (1분)**
- 시각장애인 페르소나: 폰 3번 흔들기 → 진동 + 음성 응답
- "도움이 필요하시군요. 가족에게 연결합니다."

**③ 발달장애 시나리오 (1분)**
- 그림카드 4장: 🆘 (도움) / ⚠️ (사기 같음) / ❓ (혼란) / 🚨 (긴급)
- "사기 같음" 탭 → 보호자 알림 + AI 위시퍼 응대

**④ 마무리 (1분)**
- 화면에 ROI 한 줄: "**보이스피싱 2,800억 부담 vs 시스템 50억 — 56배 ROI**"
