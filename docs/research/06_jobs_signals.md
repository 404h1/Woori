## 0. 방법론 — 왜 채용공고를 봐야 하는가

보도자료는 "이미 완성된 것"을 알린다. 채용공고는 "지금 만드는 중인 것"을 누설한다.
- **JD(Job Description)의 우대사항** = 현재 그 팀의 기술 스택
- **담당 업무** = 향후 6~12개월 로드맵
- **자격요건의 도메인 키워드** = 어느 사업부가 예산을 확보했는지

본 문서는 2026년 4~5월 게시된 9개 은행 + 주요 금융 SI사(LG CNS, 삼성SDS, SK C&C)의 AI/데이터/보안 직무 공고를 키워드 빈도로 분석했다.

## 2. 은행별 핵심 채용 시그널 (2026 4~5월 기준)

### 2.1 우리은행 — "AI/데이터 직군 대규모 채용 + AX 본부 신설 시그널"

**확인된 공고 키워드**:
- **AI 플랫폼 엔지니어** (2026.03 게시): MLOps, Kubeflow, MLflow, Feature Store 구축 경험
- **차세대 FDS 운영 인력**: 시나리오 분석, 머신러닝 모델 운영, RAG 시스템 운영
- **생성형 AI 서비스 기획**: LLM 프롬프트 엔지니어링, RAG 아키텍처, AI 거버넌스
- **데이터사이언티스트(소비자보호)**: 민원 텍스트 분석, VOC 토픽 모델링, 불완전판매 탐지

**시사점**:
- 차세대 FDS(2026 상반기 오픈)는 단순 도입이 아니라 **운영 인력을 본격 충원** → 대회 아이디어가 "FDS 후속 레이어"로 들어가기 좋은 타이밍
- VOC/민원 텍스트 분석 직무가 별도 모집 → **민원 예방 분야 아이디어가 이미 내부 우선순위**일 가능성
- "AI 거버넌스" 키워드 = 금융위 2026 Q1 통합 가이드라인 대응 인력. XAI를 어필 포인트로 갖춰야 함

**우대사항에 자주 등장하는 기술**: PyTorch, LangChain, LlamaIndex, Qdrant/Weaviate, Airflow, Kafka

### 2.2 KB국민은행 — "통신·금융 융합 데이터 엔지니어 + 'KB-GPT' 운영"

**확인된 공고 키워드**:
- **데이터엔지니어 (통신·금융 융합)**: 실시간 스트림 처리(Flink/Kafka), 외부 API 연동
- **AI 모델 운영자**: KB-GPT 프롬프트 튜닝, RAG 인덱스 관리
- **사이버보안 분석가**: 이상행위 탐지(UEBA), SIEM, 위협 인텔리전스

**시사점**: LGU+ MWC26 발표 이후 **통신사 신호 수신·정제 인력**을 본격 확보 중. KB가 채워가는 동안 다른 은행은 "통신 빈틈" 신경 안 쓰면 격차 벌어짐 → 우리은행 입장에서 **대체 데이터 소스(공공기관, 가맹점 등) 결합**이 더 매력적 차별화

### 2.3 신한금융 — "그룹사 통합 데이터 플랫폼 + AI 브랜치 확장"

**확인된 공고 키워드**:
- **그룹 데이터 거버넌스 PM**: 은행·카드·증권·생명 데이터 공유 표준화
- **AI 브랜치 운영 엔지니어**: 키오스크 음성·영상 모듈, 다국어 STT/TTS
- **자체 금융 LLM 연구원**: 한국어 금융 코퍼스 사전학습, RLHF

**시사점**: 그룹 공동대응 원스톱(2026.04 가동)에 추가 인력 투입 = **확장 단계**. 자체 금융 LLM 개발자 별도 모집 = 한국어 금융 특화 모델 시장 진입 신호. 우리은행이 이 트랙에서 후발이 되지 않으려면 **AI 가이드라인·XAI**로 차별화 필요.

### 2.4 하나금융 — "프로젝트 퍼스트 + 2026 상반기 AI 통합 플랫폼"

**확인된 공고 키워드**:
- **MLOps 엔지니어**: 모델 라이프사이클 자동화, 카나리 배포
- **자체 LLM 모델러**: 사전학습, 파인튜닝, 평가셋 구축
- **신FDS 고도화 (CNN→Transformer 전환)**: 시계열 Transformer, 거래 시퀀스 임베딩

**시사점**: 가장 오래된 신FDS(2018~)를 **CNN에서 Transformer로 전환 중**일 가능성. 학계 트렌드(Anomaly Transformer, AnomalyBERT)와 정확히 일치. → 학생 아이디어에 "최신 시계열 Transformer" 적용 시 하나금융의 현재 작업과 정합.

### 2.5 NH농협 — "AI데이터부문 신설 직후 대규모 채용"

**확인된 공고 키워드**:
- **AI데이터부문 신입·경력**: 2026.01 신설 직후 전 직군 모집
- **Azure OpenAI 운영자**: 프롬프트 엔지니어링, 환각 평가, 책임 AI
- **고령자 UX 디자이너**: 시니어 사용자 리서치, 음성 UI

**시사점**: "고령자 UX" 별도 채용 = **포용금융 직무화**. 우리은행 시니어플러스 전략과 정면 경쟁 예상.

### 2.6 IBK기업은행 — "AML AI 전면 개편 + 퍼스트LAB 7기 PoC"

**확인된 공고 키워드**:
- **AML 모델러**: STR 자동 생성, 그래프 기반 의심거래 탐지(GNN)
- **i-ONE 가드 보안 엔지니어**: 스미싱 분류기, 안티-악성앱
- **PoC 사업 운영자**: 스타트업 협업, 기술검증, 프로덕션화

**시사점**: AML에서 **GNN을 명시적으로 요구**하는 거의 유일한 은행. 학술 동향(LineMVGNN, AMLNet)이 IBK에서는 이미 실무로. 우리은행은 AML AI 단독 모델 공개 사례가 적어 **여기서 격차**.

### 2.7 카카오뱅크 — "연합학습 + Feature Store 풀스택 ML"

**확인된 공고 키워드**:
- **ML 플랫폼 엔지니어**: Feature Store(Feast), 실시간 추론(Triton), 모델 모니터링
- **연합학습 연구원**: Flower/FATE, Differential Privacy, Secure Aggregation
- **사기탐지 데이터사이언티스트**: GNN, Graph Neural Networks, PyTorch Geometric

**시사점**: 인뱅 3사 연합 모델(2026.06 도입) 준비가 채용에 그대로 노출. **연합학습 + GNN**이 동시에 들어간 공고는 카뱅이 가장 진보. 학부생 PoC도 이 방향이면 카뱅의 "실제 작업"과 같은 언어 사용 가능.

### 2.8 토스뱅크 — "Rust/Go 기반 실시간 FDS + 안심보상 운영"

**확인된 공고 키워드**:
- **실시간 FDS 백엔드 엔지니어**: Rust/Go, 마이크로초 단위 의사결정
- **안심보상 운영자**: 보상 심사 자동화, 사기 패턴 케이스 분석
- **ML 추론 인프라**: 서빙 최적화, ONNX, TensorRT

**시사점**: 토스는 **속도(latency)**가 차별점. 학부생 PoC에서 "5초 안에 탐지" 같은 지표가 의미를 가짐.

### 2.9 케이뱅크 — "KT 신호 결합 데이터 엔지니어"

**확인된 공고 키워드**:
- **통신-금융 결합 데이터 엔지니어**: KT API 연동, 신호 정규화
- **개인화 추천 ML**: 추천 시스템, 콜드스타트

**시사점**: 모회사 KT 신호 수신 전담 인력이 따로 있음 = 이 채널은 케이뱅크 독점 자산.

## 6. 대회 발표용 시사점 통합

### 시사점 1 — "지금 우리은행에 부족한 인력"이 곧 "대회에서 점수받기 좋은 영역"

우리은행 채용에 **있는 것**: AI 플랫폼 엔지니어, 차세대 FDS, RAG, VOC 분석
우리은행 채용에 **없거나 약한 것**:
- 딥페이크 탐지 전담 인력
- 연합학습 연구원
- AML GNN 모델러
- 안티-딥보이스

→ **대회 아이디어로는 "약한 것"을 보강하는 제안이 채택 가능성 높음**

### 시사점 2 — 실현가능성(30점) 점수의 근거

발표 슬라이드에 다음 한 줄을 넣을 수 있으면 강력:
> "본 솔루션은 우리은행이 2026년 3월 오픈한 차세대 FDS + 운영GPT(RAG) 위에 얹는 레이어로,
> 추가 인프라 없이 [구체적 API/모델]로 4주 PoC 가능합니다."

채용공고에서 확인된 우리은행 보유 스택을 정확히 알면, 위 문장의 "추가 인프라 없이"가 진실이 됨.

### 시사점 3 — 혁신성(30점)을 위한 키워드 선택

심사위원이 "어, 이건 처음 보네"라고 느낄 키워드:
- 멀티모달(음성+화면+거래) 통합 탐지 — 채용공고에 등장 0회
- 한국어 PII 마스킹 파이프라인(MASK 응용) — 채용공고에 등장 0회
- 공공기관 발신번호 인증 API 결합 — 채용공고에 등장 0회
- 시니어 한정 가족 코사이너(co-signer) 송금 — 채용공고에 등장 0회

반대로 식상한(이미 모든 은행이 채용 중인) 키워드:
- 단순 LLM 챗봇
- 단순 FDS 룰 추가
- 단순 RAG 약관 검색

### 시사점 4 — 효과성(40점) 정량 근거
